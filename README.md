# Le Hangar à Houblon — Maquette du site

**Maquette du futur site web** du bar *Le Hangar à Houblon* (Suippes).  
Cette maquette est mise à disposition **pour consultation par les clients** avant le prochain rendez-vous.

---

## Voir la maquette en ligne (GitHub)

Une fois le dépôt publié sur GitHub, vous pouvez consulter la maquette directement dans le navigateur :

- **Via GitHub Pages** (si activé) : ouvrez le lien indiqué en haut à droite du dépôt (*« About »* → *« Website »*) ou le lien fourni par votre interlocuteur.
- **Sans compte GitHub** : aucun compte n’est nécessaire pour voir la maquette ; un simple lien suffit.

La maquette se comporte comme un site : défilement horizontal, menu, sections (accueil, le lieu, ambiance, soirées, privatiser, infos). Idéal pour la regarder sur ordinateur ou téléphone avant notre prochain rdv.

---

## Consulter la maquette en local (téléchargement)

Si vous préférez **télécharger le projet** (dossier ou archive ZIP) pour l’ouvrir sur votre ordinateur ou votre téléphone, sans être connecté à GitHub :

**En bref :**
- **Ordinateur** : ouvrez le dossier → double-cliquez sur **index.html** → la maquette s’ouvre dans votre navigateur.
- **Téléphone** : ouvrez d’abord l’archive ZIP pour obtenir le dossier → ouvrez le fichier **index.html** avec Chrome ou Safari.

**Important :** ne déplacez pas les fichiers et ne supprimez rien. Tout doit rester dans le même dossier (y compris le dossier **assets** et les images).

---

## Fichiers du projet

| Emplacement | Fichiers |
|------------|----------|
| **À la racine** | `index.html`, `styles.css`, `script.js`, `README.md`, `README.txt` |
| **Dossier `assets`** | `logo.png`, `hero-convivial.jpg`, images de la galerie (`gallery-*.jpg`), `bento-convivial.jpg` |

Si un fichier ou le dossier **assets** manque, la maquette ne s’affichera pas correctement.

---

## Utilisation de la maquette (une fois ouverte)

- **Ordinateur** : molette pour passer d’une section à l’autre, ou glisser la page ; les points à droite permettent de sauter à une section.
- **Téléphone** : glisser le doigt gauche/droite ; le menu (☰) en haut à droite ouvre la liste des sections.

Une connexion internet (Wi‑Fi ou données) est recommandée pour que les polices et la carte s’affichent correctement.

---

## Dépannage

| Problème | À vérifier |
|----------|------------|
| Couleurs ou images manquantes | Tous les fichiers et le dossier **assets** doivent rester au même endroit. |
| Pas de fichier `index.html` sur téléphone | Ouvrez d’abord l’archive ZIP pour obtenir le dossier. |
| Carte ou textes incorrects | Connectez-vous au Wi‑Fi ou activez les données mobiles. |
| Quel logiciel pour ouvrir ? | Utilisez un navigateur (Chrome, Safari, Edge, etc.). |
