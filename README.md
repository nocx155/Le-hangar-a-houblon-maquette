# Comment regarder la maquette du site (Le Hangar à Houblon)

**Pour lire ce mode d’emploi directement dans le Bloc-notes** (sans avoir à choisir avec quoi l’ouvrir), ouvrez le fichier **README.txt** à la place de ce fichier.

**Pas besoin de s’y connaître en informatique.** Suivez les étapes ci-dessous, tout est expliqué simplement.

Vous avez reçu un **concept du futur site** : vous pouvez l’ouvrir sur votre ordinateur ou sur votre téléphone, comme un site internet, **sans rien publier en ligne**. Tout reste chez vous.

---

## En deux mots

- **Sur ordinateur** : ouvrez le dossier que vous avez reçu → double-cliquez sur le fichier **index.html** → le site s’ouvre dans votre navigateur.
- **Sur téléphone** : si vous avez reçu une « archive » (fichier zip), il faut d’abord l’**ouvrir** pour obtenir un dossier → puis touchez le fichier **index.html** et ouvrez-le avec Chrome ou Safari.

Le plus important : **ne déplacez pas les fichiers tout seuls** et **ne supprimez rien**. Tout doit rester dans le même dossier (le fichier index.html, les autres fichiers, et le dossier « assets » avec les photos).

---

## Vérifier que vous avez bien tous les fichiers

Quand vous ouvrez le dossier (ou après avoir ouvert le zip), vous devez voir **exactement** ceci :

**À la racine du dossier :**
- **index.html** — le fichier à ouvrir pour voir le site
- **styles.css** — mise en forme du site
- **script.js** — menu et défilement
- **README.txt** — ce mode d’emploi (s’ouvre avec le Bloc-notes en double-cliquant)
- **README.md** — même contenu en version « technique » (optionnel)

**Un dossier nommé « assets »** (à l’intérieur, les images) :
- **logo.png**
- **hero-convivial.jpg**
- **gallery-soiree-convivial.jpg**
- **gallery-dj-convivial.jpg**
- **gallery-concert-convivial.jpg**
- **gallery-bieres-convivial.jpg**
- **gallery-comptoir-convivial.jpg**
- **bento-convivial.jpg**

Si un de ces fichiers ou le dossier **assets** manque, la maquette ne s’affichera pas correctement (images ou couleurs manquantes). Dans ce cas, redemandez l’envoi complet du dossier.

## Sur un ordinateur

### Vous avez reçu un dossier (plusieurs fichiers ensemble)

1. Ouvrez ce dossier (double-clic avec la souris).
2. Cherchez le fichier qui s’appelle **index.html** (il a souvent une icône de navigateur ou de page).
3. Double-cliquez sur **index.html**.
4. Une page internet s’ouvre : c’est la maquette du site.

### Vous avez reçu un seul fichier qui se termine par « .zip »

Un fichier « zip » est une **boîte** qui contient le dossier. Il faut d’abord l’ouvrir pour voir les fichiers à l’intérieur.

**Sur Windows :**
1. Faites un **clic droit** sur le fichier zip.
2. Cliquez sur **« Extraire tout »** (ou « Décompresser »).
3. Choisissez un endroit (par exemple le Bureau) et validez.
4. Un **nouveau dossier** apparaît. Ouvrez-le.
5. Dans ce dossier, double-cliquez sur le fichier **index.html**.
6. La maquette s’ouvre dans votre navigateur.

**Sur Mac :**
1. Double-cliquez sur le fichier zip.
2. Un **dossier** apparaît au même endroit. Ouvrez-le.
3. Double-cliquez sur le fichier **index.html**.
4. La maquette s’ouvre dans votre navigateur.

## Sur un téléphone

### Étape 1 : Récupérer le fichier sur le téléphone

- Si on vous a envoyé un **lien** (WeTransfer, Google Drive, etc.) : ouvrez le lien sur votre téléphone et téléchargez le fichier (souvent un fichier qui se termine par **.zip**).
- Si c’est par **e-mail** : ouvrez la pièce jointe et téléchargez-la.

### Étape 2 : Ouvrir le fichier zip (pour obtenir le dossier)

Le fichier zip est comme une **boîte** : il faut l’ouvrir pour voir le contenu.

**Sur Android :**
1. Ouvrez l’application **Fichiers** (ou **Mes fichiers**).
2. Allez dans **Téléchargements** (ou l’endroit où le fichier a été enregistré).
3. Repérez le fichier zip (nom qui se termine par **.zip**).
4. **Appuyez longtemps** (environ 2 secondes) sur ce fichier.
5. Un menu s’affiche : choisissez **« Extraire »** ou **« Décompresser »**.
6. Un **dossier** est créé. C’est dans ce dossier qu’il faut aller à l’étape suivante.

**Sur iPhone ou iPad :**
1. Ouvrez l’application **Fichiers**.
2. Allez dans **Téléchargements** (ou l’endroit où se trouve le fichier).
3. Appuyez sur le fichier zip. Souvent, le téléphone ouvre tout seul un dossier.
4. Si rien ne s’ouvre : vous pouvez installer une application gratuite comme **« Documents »** ou **« RAR »** pour ouvrir les fichiers zip.

### Étape 3 : Ouvrir la maquette (le site)

**Sur Android :**
1. Ouvrez le **dossier** que vous venez d’obtenir (dans l’app Fichiers).
2. Cherchez le fichier **index.html**.
3. Appuyez dessus.
4. Le téléphone demande « Avec quelle application ouvrir ? » → choisissez **Chrome** (ou le navigateur que vous utilisez d’habitude).
5. La maquette du site s’affiche comme une page internet.

**Sur iPhone ou iPad :**
1. Dans l’app **Fichiers**, ouvrez le dossier qui contient la maquette.
2. Cherchez le fichier **index.html** et appuyez dessus.
3. Si la page s’affiche dans Fichiers : appuyez sur l’icône **Partager** (carré avec une flèche vers le haut) → puis **« Ouvrir dans Safari »** (ou Chrome).
4. La maquette s’affiche comme une page internet.

## Une fois la maquette ouverte : comment s’en servir ?

Le site défile **horizontalement** : chaque « page » est une section (accueil, le lieu, ambiance, soirées, privatiser, infos).

**Sur ordinateur :**
- **Molette de la souris** : tournez la molette vers le bas pour aller à la section suivante, vers le haut pour la précédente. Le défilement se fait **une section à la fois**, avec une transition fluide.
- Vous pouvez aussi **faire glisser la page** vers la gauche ou la droite (clic maintenu + déplacer la souris, ou glisser avec le trackpad).
- Les **petits points à droite** permettent de sauter directement à une section.

**Sur téléphone :**
- **Glissez le doigt** vers la gauche ou la droite pour passer d’une section à l’autre.
- Le **menu (trois barres ☰)** en haut à droite ouvre la liste des sections ; touchez une section pour y aller.

**Connexion internet :** pour que les polices et la carte s’affichent comme prévu, il vaut mieux être connecté au Wi-Fi ou aux données. Sans internet, le texte et les images s’affichent quand même, mais l’apparence peut être un peu différente.

## Ça ne marche pas ? Voici quoi vérifier

**La page s’ouvre mais il manque les couleurs, les images ou la mise en page :**
- Tous les fichiers doivent être **au même endroit** : le fichier index.html, les autres fichiers du dossier, et le dossier **assets** (avec les photos dedans). Si vous avez déplacé ou supprimé quelque chose, remettez tout ensemble comme au départ.

**Sur téléphone, je ne trouve pas le fichier index.html :**
- Assurez-vous d’avoir bien **ouvert le fichier zip** (étape 2). Le fichier index.html est **à l’intérieur** du dossier qui apparaît après avoir ouvert le zip.

**La carte ou les textes ne s’affichent pas bien :**
- Connectez votre téléphone ou votre ordinateur au **Wi-Fi** ou activez les **données mobiles**. Certains éléments se chargent depuis internet.

**Je ne sais pas avec quelle application ouvrir le fichier :**
- Choisissez **Chrome** ou **Safari** (ou tout autre navigateur que vous utilisez pour aller sur internet). C’est comme ouvrir une page web.

2. **Ordinateur** : ouvrir le dossier → double-clic sur **index.html**.
3. **Téléphone** : ouvrir le fichier zip pour avoir le dossier → toucher **index.html** → l’ouvrir avec Chrome ou Safari.

Aucun site n’est publié sur internet : tout reste sur votre ordinateur ou votre téléphone.
