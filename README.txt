================================================================================
  LE HANGAR À HOUBLON — MAQUETTE DU SITE
================================================================================

Maquette du futur site web du bar Le Hangar à Houblon (Suippes).
Cette maquette est mise à disposition pour consultation par les clients
avant le prochain rendez-vous.


--------------------------------------------------------------------------------
VOIR LA MAQUETTE EN LIGNE (GITHUB)
--------------------------------------------------------------------------------

Si le projet a été mis en ligne sur GitHub, vous pouvez consulter la maquette
directement dans votre navigateur, sans rien télécharger :

  - Ouvrez le lien fourni (par e-mail ou par votre interlocuteur).
  - Aucun compte GitHub n'est nécessaire.
  - La maquette se comporte comme un site : défilement horizontal, menu,
    sections (accueil, le lieu, ambiance, soirées, privatiser, infos).

Idéal pour la regarder sur ordinateur ou téléphone avant le prochain rdv.


--------------------------------------------------------------------------------
CONSULTER EN LOCAL (APRÈS TÉLÉCHARGEMENT)
--------------------------------------------------------------------------------

Si vous avez téléchargé le dossier ou une archive .zip, vous pouvez ouvrir
la maquette sur votre ordinateur ou votre téléphone.

  Sur ordinateur : ouvrez le dossier, puis double-cliquez sur "index.html".
  Sur téléphone  : ouvrez d'abord le fichier .zip pour obtenir le dossier,
                   puis ouvrez "index.html" avec Chrome ou Safari.

Important : ne déplacez pas les fichiers et ne supprimez rien. Tout doit
rester dans le même dossier (index.html, les autres fichiers, et le dossier
"assets" avec les images).


--------------------------------------------------------------------------------
VÉRIFIER QUE VOUS AVEZ TOUS LES FICHIERS
--------------------------------------------------------------------------------

À la racine du dossier :
  - index.html     (fichier à ouvrir pour voir le site)
  - styles.css     (mise en forme)
  - script.js      (menu et défilement)
  - README.txt     (ce mode d'emploi)

Dossier "assets" (images) :
  - logo.png
  - hero-convivial.jpg
  - gallery-soiree-convivial.jpg, gallery-dj-convivial.jpg,
    gallery-concert-convivial.jpg, gallery-bieres-convivial.jpg,
    gallery-comptoir-convivial.jpg
  - bento-convivial.jpg

Si un fichier ou le dossier "assets" manque, la maquette ne s'affichera
pas correctement.


--------------------------------------------------------------------------------
SUR UN ORDINATEUR (FICHIER .ZIP)
--------------------------------------------------------------------------------

  1. Clic droit sur le fichier .zip → "Extraire tout" (Windows)
     ou double-clic sur le .zip (Mac).
  2. Ouvrez le dossier créé.
  3. Double-cliquez sur "index.html".
  4. La maquette s'ouvre dans votre navigateur.


--------------------------------------------------------------------------------
SUR UN TÉLÉPHONE
--------------------------------------------------------------------------------

  1. Téléchargez le fichier (lien ou pièce jointe).
  2. Ouvrez le fichier .zip (Extraction / Décompresser).
  3. Dans le dossier obtenu, ouvrez "index.html" avec Chrome ou Safari.


--------------------------------------------------------------------------------
UTILISATION DE LA MAQUETTE
--------------------------------------------------------------------------------

  Ordinateur : molette pour changer de section ; points à droite pour sauter
               à une section ; menu (trois barres) pour la liste.
  Téléphone  : glisser le doigt gauche/droite ; menu en haut à droite.

Connexion internet recommandée pour les polices et la carte.


--------------------------------------------------------------------------------
ÇA NE MARCHE PAS ?
--------------------------------------------------------------------------------

  - Couleurs/images manquantes : ne déplacez rien ; tout doit rester au même endroit.
  - Sur téléphone, pas d'index.html : ouvrez d'abord le fichier .zip.
  - Carte ou textes bizarres : connectez-vous au Wi-Fi ou aux données.
  - Quel logiciel ? : Chrome, Safari ou tout navigateur internet.
