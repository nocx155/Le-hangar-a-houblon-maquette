================================================================================
  COMMENT REGARDER LA MAQUETTE DU SITE (LE HANGAR À HOUBLON)
================================================================================

Pas besoin de s'y connaître en informatique. Suivez les étapes ci-dessous, 
tout est expliqué simplement.

Vous avez reçu un concept du futur site : vous pouvez l'ouvrir sur votre 
ordinateur ou sur votre téléphone, comme un site internet, sans rien 
publier en ligne. Tout reste chez vous.


--------------------------------------------------------------------------------
EN DEUX MOTS
--------------------------------------------------------------------------------

  Sur ordinateur : ouvrez le dossier que vous avez reçu, puis double-cliquez 
  sur le fichier "index.html". Le site s'ouvre dans votre navigateur.

  Sur téléphone : si vous avez reçu une archive (fichier .zip), il faut 
  d'abord l'ouvrir pour obtenir un dossier, puis touchez le fichier 
  "index.html" et ouvrez-le avec Chrome ou Safari.

Le plus important : ne déplacez pas les fichiers tout seuls et ne supprimez 
rien. Tout doit rester dans le même dossier (index.html, les autres fichiers, 
et le dossier "assets" avec les photos).


--------------------------------------------------------------------------------
VÉRIFIER QUE VOUS AVEZ BIEN TOUS LES FICHIERS
--------------------------------------------------------------------------------

Quand vous ouvrez le dossier (ou après avoir ouvert le zip), vous devez 
voir exactement ceci :

  À la racine du dossier :
    - index.html     (le fichier à ouvrir pour voir le site)
    - styles.css     (mise en forme du site)
    - script.js      (menu et défilement)
    - README.txt     (ce mode d'emploi)

  Un dossier nommé "assets" (à l'intérieur, les images) :
    - logo.png
    - hero-convivial.jpg
    - gallery-soiree-convivial.jpg
    - gallery-dj-convivial.jpg
    - gallery-concert-convivial.jpg
    - gallery-bieres-convivial.jpg
    - gallery-comptoir-convivial.jpg
    - bento-convivial.jpg

Si un de ces fichiers ou le dossier "assets" manque, la maquette ne 
s'affichera pas correctement. Dans ce cas, redemandez l'envoi complet du dossier.


--------------------------------------------------------------------------------
SUR UN ORDINATEUR
--------------------------------------------------------------------------------

Vous avez reçu un dossier (plusieurs fichiers ensemble) :
  1. Ouvrez ce dossier (double-clic avec la souris).
  2. Cherchez le fichier "index.html".
  3. Double-cliquez sur index.html.
  4. Une page internet s'ouvre : c'est la maquette du site.

Vous avez reçu un seul fichier qui se termine par .zip :
  Un fichier zip est une boîte qui contient le dossier. Il faut d'abord 
  l'ouvrir pour voir les fichiers à l'intérieur.

  Sur Windows :
    1. Clic droit sur le fichier zip.
    2. Cliquez sur "Extraire tout" (ou "Décompresser").
    3. Choisissez un endroit (par exemple le Bureau) et validez.
    4. Un nouveau dossier apparaît. Ouvrez-le.
    5. Dans ce dossier, double-cliquez sur "index.html".
    6. La maquette s'ouvre dans votre navigateur.

  Sur Mac :
    1. Double-cliquez sur le fichier zip.
    2. Un dossier apparaît au même endroit. Ouvrez-le.
    3. Double-cliquez sur le fichier "index.html".
    4. La maquette s'ouvre dans votre navigateur.


--------------------------------------------------------------------------------
SUR UN TÉLÉPHONE
--------------------------------------------------------------------------------

Étape 1 - Récupérer le fichier sur le téléphone :
  - Lien (WeTransfer, Google Drive, etc.) : ouvrez le lien et téléchargez 
    le fichier (souvent un .zip).
  - E-mail : ouvrez la pièce jointe et téléchargez-la.

Étape 2 - Ouvrir le fichier zip :
  Le fichier zip est comme une boîte : il faut l'ouvrir pour voir le contenu.

  Sur Android :
    1. Ouvrez l'application "Fichiers" (ou "Mes fichiers").
    2. Allez dans "Téléchargements".
    3. Repérez le fichier .zip.
    4. Appuyez longtemps (environ 2 secondes) sur ce fichier.
    5. Choisissez "Extraire" ou "Décompresser".
    6. Un dossier est créé. C'est dans ce dossier qu'il faut aller à l'étape 3.

  Sur iPhone ou iPad :
    1. Ouvrez l'application "Fichiers".
    2. Allez dans "Téléchargements".
    3. Appuyez sur le fichier zip. Souvent, le téléphone ouvre tout seul un dossier.
    4. Si rien ne s'ouvre : installez une app gratuite comme "Documents" ou "RAR".

Étape 3 - Ouvrir la maquette :
  Sur Android :
    1. Ouvrez le dossier que vous venez d'obtenir.
    2. Cherchez le fichier "index.html".
    3. Appuyez dessus.
    4. Choisissez "Ouvrir avec" puis "Chrome" (ou votre navigateur).
    5. La maquette s'affiche comme une page internet.

  Sur iPhone ou iPad :
    1. Dans l'app Fichiers, ouvrez le dossier qui contient la maquette.
    2. Cherchez "index.html" et appuyez dessus.
    3. Si besoin : icône Partager (carré avec flèche) puis "Ouvrir dans Safari" ou Chrome.
    4. La maquette s'affiche comme une page internet.


--------------------------------------------------------------------------------
UNE FOIS LA MAQUETTE OUVERTE : COMMENT S'EN SERVIR ?
--------------------------------------------------------------------------------

Le site défile horizontalement : chaque "page" est une section (accueil, 
le lieu, ambiance, soirées, privatiser, infos).

  Sur ordinateur :
    - Molette de la souris : vers le bas = section suivante, vers le haut = 
      section précédente. Une section à la fois, avec une transition fluide.
    - Vous pouvez aussi faire glisser la page vers la gauche ou la droite.
    - Les petits points à droite permettent de sauter à une section.

  Sur téléphone :
    - Glissez le doigt vers la gauche ou la droite pour changer de section.
    - Le menu (trois barres) en haut à droite ouvre la liste des sections.

  Connexion internet : pour que les polices et la carte s'affichent bien, 
  connectez-vous au Wi-Fi ou aux données. Sans internet, le texte et les 
  images s'affichent quand même, mais l'apparence peut être un peu différente.


--------------------------------------------------------------------------------
ÇA NE MARCHE PAS ? VOICI QUOI VÉRIFIER
--------------------------------------------------------------------------------

  La page s'ouvre mais il manque les couleurs ou les images :
    Tous les fichiers doivent être au même endroit. Ne déplacez rien.
    Remettez tout ensemble comme au départ si besoin.

  Sur téléphone, je ne trouve pas index.html :
    Assurez-vous d'avoir bien ouvert le fichier zip (étape 2). Le fichier 
    index.html est à l'intérieur du dossier qui apparaît après avoir ouvert le zip.

  La carte ou les textes ne s'affichent pas bien :
    Connectez-vous au Wi-Fi ou activez les données mobiles.

  Je ne sais pas avec quelle application ouvrir le fichier :
    Choisissez Chrome ou Safari (ou tout navigateur pour aller sur internet).
